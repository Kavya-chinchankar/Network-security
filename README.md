# 30-Day Network Engineer + Network Security Road

**Target Roles:** Network Engineer, Network Security Engineer, SOC/NOC, Platform Security Engineer
**Study Commitment:** 4 Hours / Day
**Outcome:** Interview‑ready networking fundamentals with security depth and hands‑on confidence

---

## 📌 Repository Purpose

This repository documents a **day‑by‑day, industry‑aligned learning journey** covering **networking fundamentals, routing & switching, network security, monitoring, and interview readiness**.

Each day includes:

* Concept notes
* Diagrams / explanations
* Hands‑on labs or simulations
* Security perspective
* Interview questions

The repository is structured to show **consistent daily progress**, suitable for **recruiter review, interview discussion, and self‑revision**.

---

## 🧭 Study Model Followed

**Concept → Configuration → Security → Troubleshooting**

Daily time split:

* 1.5 hrs – Theory & diagrams
* 1.5 hrs – Hands‑on / labs
* 1 hr – Security + interview prep
